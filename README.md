# News Aggregator App

This is a Flask web application that fetches and displays the latest news articles using News API.

## Features
- Display latest news
- Show article title and content
- Link to original source

## How to run the project

1. Install the requirements

pip install -r requirements.txt

2. Create a .env file and add your API key

NEWS_API_KEY=your_api_key_here

3. Run the application

python app.py

4. Open in your browser

http://127.0.0.1:5000/

## Note
Do not upload the .env file to GitHub because it contains your API key.
